# Contributing to OpenCurrencyX

Thank you for your interest in contributing! Please read these guidelines before submitting a pull request or issue.

## How to Contribute
- Fork the repository and create your branch from `main`.
- Write clear, concise commit messages.
- Add tests for new features and bug fixes.
- Ensure all checks pass before submitting a PR.

## Code Style
- Use Prettier and ESLint for TypeScript/JS code.
- Follow PEP8 for Python code.

## Reporting Issues
- Use the issue templates provided.
- Include steps to reproduce, expected and actual behavior.

## Community
- Be respectful and inclusive.
- See our Code of Conduct.
